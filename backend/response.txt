# 2Ô∏è‚É£ **READ (GET)**

## ‚úî GET ALL posts

```ts
const posts = await Post.findAll();
```

## ‚úî GET one post by slug

```ts
const post = await Post.findOne({ where: { slug: req.params.slug }});
```

## ‚úî GET post by id

```ts
const post = await Post.findByPk(12);
```

## ‚úî GET comments for a post

```ts
const comments = await Comment.findAll({
  where: { postId: req.params.postId }
});
```

## ‚úî GET comments with user + post details (JOIN)

```ts
const comments = await Comment.findAll({
  where: { postId: req.params.postId },
  include: [User, Post]
});
```

## ‚úî GET posts including comments (JOIN)

```ts

  include: [Comment]
});
```

# 3Ô∏è‚É£ **UPDATE**

## ‚úî Update a post


## ‚úî Update a comment

```ts

```
